Todoist Auto

Descrizione

  • Gabriele Pucciarelli

  • 5 settembre, 2022

Todoist Auto Ã¨ uno script per Google Assistant e Android, realizzato con Tasker, Javascript, Actions on Google e le API di Todoist. Lo script permette di invocare una Action vocale con Google Assistant (su smartphone, Wear OS, Android Auto, Google Home mini etc.) per inserire task su Todoist. A seconda della frase, i task vengono inseriti in progetti diversi o con etichette diverse (ad esempio "Ricordami prima di uscire...", "Ricordami quando torno a casa...", "Ricordami di telefonare a...". Inoltre, al collegamento con il bluetooth dell'auto, lo script legge ad alta voce tutti i task sotto al filtro "In auto" di Todoist.

Tecnologie utilizzate

  • Android
  • Tasker
  • Javascript
  • Actions on Google
  • Todoist API
